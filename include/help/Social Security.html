Overview

Use the Social Security module to track a client's estimated Social Security benefits. The Social Security Administration sends annual statements to each individual showing their estimated benefits. Use the data from the latest statement to fill this form.

Actions

Follow the instructions below to perform tasks such as creating, editing, and exporting Social Security records.

To…

Then…

Create a Social Security record
  1. Access the Social Security Create form using one of the following:
    • Click Retirement Planning > Social Security > Create Social Security from the Navigation menu.
    • Click Create New on the Social Security List View.
  2. Complete the Social Security Create Form. See Field Descriptions for information about each field. Fields that have an asterisk (*) are required; you must enter data in the field to save the record.
  3. Click Save to save the record and view its detail page, or click Cancel .

Display a list of Social Security records

Click Retirement Planning > Social Security > View Social Security.
The Social Security List View displays.

Display the details for a Social Security record

Click the name in the Social Security List View.
The Social Security Detail View displays.

Edit Social Security information
  1. Access the Social Security Edit View by doing one of the following:
    • Click Edit in the Social Security Detail View.
    • Click the edit icon (pencil) associated with the record in the Social Security List View.
  2. Update the information. See Field Descriptions for information about each field.
  3. Click Save to update the record.

Suppress one or more Social Security records
  1. In the Social Security List View, click the Selection box associated with the record(s) you want to suppress.
  2. Click Suppress.
    A confirmation box displays.
  3. Click OK.
    The records are suppressed.

You can also suppress a single record by clicking Suppress on the Social Security Detail View.

Update information related to multiple Social Security records
  1. In the Social Security List View, click the Selection box associated with the records you want to update.
  2. Click Mass Update in the Actions dropdown.

For more information, see Editing and suppressing multiple records.

Duplicate Social Security information

Click Duplicate in the Social Security Detail View. Duplication is a convenient way of creating a similar Social Security record. You can change and save the information in the duplicate record to create a new record.

Find and merge duplicate records

 Follow the instructions in Merging duplicate records.

Export records
  1. In the Social Security List View, click the Selection box associated with the records you want to export.
  2. Click Export in the dropdown and follow the instructions in Exporting data.

Field Descriptions

Fields that have an asterisk (*) are required; you must enter data in the field to save the record.

Client

Click the Client associated with the estimated Social Security benefits on the Clients Popup List View (displays upon clicking the Selection arrow) or the list of Clients matching the character(s) you enter.

Contact

Click the Contact who receives the Social Security on the Contacts Popup List View (displays upon clicking the Selection arrow) or the list of Contacts matching the character(s) you enter.

Year of Estimate

Enter the year of the latest Social Security statement.

Benefit at Age 62

Enter the annual Social Security benefit if the contact retires at age 62.

Disability

Enter the annual Social Security benefit if the contact retires with a disability.

Full Retirement Amount

Enter the maximum annual Social Security benefit.

Child

Enter the annual Social Security benefit for the contact's children.

Benefit at Age 70

Enter the annual Social Security benefit if the contact retires at age 70.

Spouse

Enter the annual Social Security benefit for the contact's spouse.

Fully Retired Spouse

Enter the maximum Social Security benefit for the contact's spouse.

Taxed Social Security Earnings

Enter the taxed Social Security earnings.

Total Family Benefits

Enter the total family benefits dollar amount .

Taxed Medicare Earnings

Enter the taxed Medicare earnings.

Description

Enter additional information about the Social Security benefits statement.
