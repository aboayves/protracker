Overview

Use the Client Types module to categorize client records. All users can view Client Types. Only users with database administration privileges can edit Client Types.

Availability

The Client Types module is available to users of the following editions:

  • Standard
  • Professional
  • Enterprise

Actions

Create a Client Type

As an Administrator, use the following instructions to create a Client Type.

  1. Click Clients > View Client Types from the Navigation menu.
    The Client Types List View displays.
  2. Click Create New.
    The Client Types Create form displays.
  3. Enter information for the fields described below. Fields that have an asterisk (*) are required; you must enter data in the field to save the Client Type.
  4. Click Save.
    The Client Types record is created.

Client Type*

Enter the name of the client type. This is a mandatory field.

Description

Enter additional details about the client type.

Assigned to

By default, the record is assigned to the creator of the Client Types record. To change the assignment, click the user on the Users Popup List View (displays upon clicking the Selection arrow) or the list of users matching the character(s) you enter.

Edit a Client Type

As an Administrator, use the following instructions to edit a Client Type.

  1. Click Clients > View Client Types from the Navigation menu.
    The Client Types List View displays.
  2. Click the edit (pencil) icon associated with the record you want to update.
    The Client Types Edit form displays.
  3. Edit the information in the fields using the field descriptions in Create a Client Type.
  4. Click Save.
    The record is updated.

Suppress a Client Type

As an Administrator, use the following instructions to suppress a Client Type.
  1. Click Clients > View Client Types from the Navigation menu.
    The Client Types List View displays.
  2. Click the selection box associated with the record(s) you want to suppress.
  3. Click Suppress in the Actions dropdown.
    The confirmation box displays.
  4. Click OK.
    The record(s) are suppressed.
