Overview

Use the Account Histories module to track changes to account values over time. Most account history records will be created by importing them from a portfolio management system. All users who can view and edit accounts can view and edit account history records. Use of the Account Histories module is not required if only the most recent account value is of interest.

Availability

The Account Histories module is available to users of the following editions:

  • Standard
  • Professional
  • Enterprise

Actions

Create an Account History Record

  1. Access the Account Histories Create form by doing one of the following:
    • Click Accounts > Account Histories > Create Account Histories on the Navigation menu.
    • Click Accounts > Accounts from the Navigation menu, click the related account to view its Account Histories sub-panel, and then click Create on the Account Histories Actions dropdown.
  2. Complete the form by entering information for the fields described below. Fields that have an asterisk (*) are required; you must enter data in the field to save the Account History.
  3. Click Save.
    The Account Histories record is added.

Account*

Click the account to which the account value applies on the Account Popup List View (displays upon clicking the Selection arrow) or the list of Accounts matching the character(s) you enter. This is a mandatory field; you must click an Account.

Value Date*

In the Date Selector click the date on which the value was accurate. This is a mandatory field; you must enter a Value Date.

Value*

Enter the value of the account as of the value date. This is a mandatory field; you must enter a Value.

Import Account History Records

  1. Click Accounts > Account Histories > Import on the Navigation menu.
    The Step1: Upload Import File displays.
  2. Follow the process described in Import data.

The import file requires the following fields as described in Create an Account History Record: Account Number, Value Date, and Value.

Edit an Account History Record

  1. Access the Account History Edit View by doing one of the following:
    • Click Accounts > Account Histories > View Account Histories from the Navigation menu and click the edit (pencil) icon on the record you want to edit.
    • Click Accounts > Accounts from the Navigation menu, click the related account to view its Account Histories sub-panel, and click Edit on the Actions dropdown.
  2. Complete the form by entering information for the form fields as described in Create an Account History Record.
  3. Click Save.
    The updates are saved to the Account Histories record.

Suppress an Account History Record

  1. Click Accounts > Account Histories > View Account Histories from the Navigation menu.
    The Account Histories List View displays.
  2. Click the selection box associated with the record(s) you want to suppress.
  3. Click Suppress in the Actions dropdown.
    The confirmation box displays.
  4. Click OK.
    The record(s) are suppressed.
