Overview

Use the Transaction Schedules module to document regular movement of money in and out of accounts such as contributions to retirement plan accounts or required withdrawals for living expenses.

Availability

The Transaction Schedules module is available to users of the following editions:

  • Standard
  • Professional
  • Enterprise

Actions

Create a Transaction Schedule record

To create a Transaction Schedule record follow these instructions.

  1. Access the Transaction Schedule Create form by doing one of the following:
    • In the Navigation menu, click Accounts > Transaction Schedule > Create Transaction Schedule.
    • In the Navigation menu, click Accounts > Accounts > View Accounts , click the Account, and then click Create on the Transaction Schedule sub-panel Action dropdown.
  2. Enter information for the fields described below. Fields that have an asterisk (*) are required; you must enter data in the field to save the document.
  3. Click Save to create the record; click Cancel to return to the List View without saving the record.

Account

Click the account related to this Transaction Schedule on the Accounts Popup List View (displays upon clicking the Selection arrow) or the list of Accounts matching the character(s) you enter.

Assigned to

By default, the record is assigned to the creator of the Transaction Schedule record. To change the assignment, click the user on the Users Popup List View (displays upon clicking the Selection arrow) or the list of users matching the character(s) you enter.

Transaction Type

Click the type of transaction that will occur.

Transaction Frequency

Click how often the transaction occurs.

Start Date

Click on the Date Selector the first date of the transaction schedule.

Next Transaction

Click on the Date Selector the date of the next transaction.

To Account

Enter details about the destination account if money was moved from one account to another.

Related Account Number

Reenter the related account number if money was moved from one account to another.

Description

Enter additional details about the Transaction Schedule.

Teams*

By default, your private team is assigned as the primary team. To change the primary team assignment, click the team on the Teams Popup List View (displays upon clicking the Selection arrow) or the list of teams matching the character(s) you enter.

To assign additional teams to the Transaction Schedule, click add (+) and click the team as described above. For more information on assigning teams, see Assigning Users and Teams to Records.

This is a mandatory field.

Edit a Transaction Schedule record

Use the following instructions to update a Transaction Schedule record.

  1. Access the Transaction Schedule Edit View by doing one of the following:
    • In the Navigation menu, click Accounts > Transaction Schedule > View Transaction Schedule, and click the edit (pencil) icon associated with the record you want to change.
      The Transaction Schedule Edit View displays.
    • In the Navigation menu, click Accounts > Accounts > View Accounts , click the Account, and then click Edit on the Transaction Schedule sub-panel Action dropdown.
  2. Update information for the fields described in Create a Transaction Schedule record.
  3. Click Save to update the record; click Cancel to return to the List View without saving the record.

Suppress a Transaction Schedule record

Suppress Transaction Schedule record(s) using the Transaction Schedule List View. For more information, see Editing and suppressing multiple records.

  1. In the Navigation menu, click Accounts > Transaction Schedule > View Transaction Schedule.
    The Transaction Schedule List View displays.
  2. In the Transaction Schedule List View, click the selection box associated with the record(s) you want to suppress.
  3. Click Suppress.
    A confirmation box displays.
  4. Click OK.
    The records are suppressed.

Alternatively, to suppress a single record, you can click Suppress on the Transactions Schedule Detail View.
