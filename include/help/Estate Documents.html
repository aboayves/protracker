Overview

Use the Estate Documents module to capture information about a client’s Estate Documents and link to electronic copies, if available. This module can store documents such as:

  • Advance Healthcare Directive
  • Deed of Gift
  • Durable Power of Attorney
  • Durable Power of Attorney for Health
  • Healthcare Proxy
  • Letter of Instruction
  • Living Will
  • Terminal Care Document

For wills, trusts, and final arrangements, use the modules specific to those types of estate documents.

Availability

The Estate Documents module is available to users of the following editions:

  • Professional
  • Enterprise

Actions

Create an Estate Document record

To create an Estate Document record follow these instructions.

  1. Access the Estate Document Create form by doing one of the following:
    • Click Estate Planning > Create Estate Documents on the Navigation menu.
    • Click Clients > View Clients from the Navigation menu, click the related Client to view its Estate Documents sub-panel, and then click Create on the Estate Documents sub-panel Actions dropdown.
  2. Enter information for the fields described below. Fields that have an asterisk (*) are required; you must enter data in the field to save the document.
  3. Click Save to create the record; click Cancel to return to the List View without saving the record.

Client

Click the client associated with the Estate Document on the Clients Popup List View (displays upon clicking the Selection arrow) or the list of Clients matching the character(s) you enter.

Contact

Click the contact associated with the Estate Document on the Contacts Popup List View (displays upon clicking the Selection arrow) or the list of Contacts matching the character(s) you enter. This is the person to whom the Estate Document applies.

Document Name*

Enter a brief description of the Estate Document. This is a mandatory field.

Estate Document Type

Click the document type.

File

Click Choose File and navigate to and select your estate document file.
The file name displays in the Select File box.

Document Date

In the Date Selector click the date the Estate Document was signed.

On File

Click the checkbox if the Estate Document is on file at the firm.

Date Recorded

Attorney

Enter the name of the attorney or law office associated with this Estate Document.

Agent

Enter the name of the person who will carry out the instructions in the estate document.

Successor Agent

Enter the name of the person who will carry out the instructions in the estate document if the primary agent is not available.

Provisions

Enter any special provisions contained in the estate document.

Assigned to

By default, the record is assigned to the creator of the NAME record. To change the assignment, click the user on the Users Popup List View (displays upon clicking the Selection arrow) or the list of users matching the character(s) you enter.

Description

Enter additional details about the estate document.

Teams*

By default, your private team is assigned as the primary team. To change the primary team assignment, click the team on the Teams Popup List View (displays upon clicking the Selection arrow) or the list of teams matching the character(s) you enter.

To assign additional teams to the Estate Document, click add (+) and click the team as described above. For more information on assigning teams, see Assigning Users and Teams to Records.

Edit an Estate Document record

Use the following instructions to make changes to an Estate Document record.

  1. Access the Estate Document Edit View by doing one of the following:
    • Click Estate Planning > View Estate Documents on the Navigation menu, and click the edit (pencil) icon associated with the Estate Document record you want to change.
    • Click Clients > View Clients from the Navigation menu, click the related Client to view its Estate Documents sub-panel, and then click Edit on the Estate Documents sub-panel Actions dropdown.
  2. Make changes to the information using the field descriptions in Create an Estate Document record.
  3. Click Save to update the record; click Cancel to return to the List View without saving the record.

Suppress an Estate Document record

Use the following instructions to suppress an Estate Document record.

  1. Click Estate Planning > View Estate Documents on the Navigation menu.
    The Estate Planning List View displays.
  2. Click the selection box associated with the record(s) you want to suppress.
  3. Click Suppress.
    A confirmation box displays.
  4. Click OK.
    The Estate Documents are removed from the Estate Documents List View.
