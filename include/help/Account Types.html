Overview

Use the Account Types module to categorize accounts. All users can view account types. Only users with database administration privileges can edit account types.

Availability

The Account Owners module is available to users of the following editions:

  • Standard
  • Professional
  • Enterprise

Actions

Create an Account Type

  1. Access the Account Type Create form by doing one of the following:
    • Click Accounts > Account Types > Create Account Type on the Navigation menu.
    • Click Accounts > Accounts from the Navigation menu, click the related account to view its Account Types sub-panel, and then click Create on the Account Types Actions dropdown.
  2. Complete the form by entering information for the fields described below. Fields that have an asterisk (*) are required; you must enter data in the field to save the Activity Type.
  3. Click Save.
    The Account Type record is added.

Account Type*

Enter the name of the Account Type. This is a mandatory field; you must enter an Account Type.

Account Category*

Click the Account Category to which this Account Type belongs from the Account Category Popup List View (displays upon clicking the Selection arrow) or the list of Account Categories matching the character(s) you enter. This is a mandatory field; you must click the Account Category.

Schedule

Click the section where this account type would appear in a financial plan.

Retirement Plan Type

Click the type of retirement plan this account type represents.

Beneficiary Required

Click the check box if this account type requires a beneficiary.

Report Sort

Enter an alphanumeric sort order for use in reports.

Description

Enter additional details about the account type.

Edit an Account Type

  1. Access the Account Type Edit View by doing one of the following:
    • Click Accounts > Account Types > View Account Types from the Navigation menu and click the edit (pencil) icon on the record you want to edit.
    • Click Accounts > Accounts from the Navigation menu, click the related account to view its Account Types sub-panel, and click Edit on the Actions dropdown.
  2. Complete the form by entering information for the form fields as described in Create an Account Type.
  3. Click Save.
    The updates are saved to the Account Type record.

Suppress an Account Type

  1. Click Accounts > Account Types > View Account Types from the Navigation menu.
    The Account Types List View displays.
  2. Click the selection box associated with the record(s) you want to suppress.
  3. Click Suppress in the Actions dropdown.
    The confirmation box displays.
  4. Click OK.
    The record(s) are suppressed.
