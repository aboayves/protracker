Overview

Use the Account Types module to categorize accounts. All users can view account types. Only users with database administration privileges can edit account types.

Actions

Follow the instructions below to perform tasks such as creating, editing, importing, and exporting Account Type records.

To…

Then…

Create a Account Types record
  1. Access the Account Types Create form by doing one of the following:
    • Click Accounts > Account Types > Create Account Types on the Navigation menu.
    • Click Accounts > Accounts from the Navigation menu, click the related account to view its Account Types sub-panel, and then click Create on the Account Types sub-panel Actions dropdown.
  2. Complete the form by entering information for the fields described in Field Definitions. Fields that have an asterisk (*) are required; you must enter data in the field to save the Account Type.
  3. Click Save.
    The Account Type record is added.

Display a list of Account Types

Click Account > Account Types > View Account Types.
The Account Types List View displays.

Display the details for a Account Type

Click the Account Type in the Account Types List View.
The Account Types Detail View displays.

Edit Account Type information
  1. Access the Account Types Edit View by doing one of the following:
    • Click Edit in the Account Types Detail View.
    • Click the edit icon (pencil) associated with the Account Type record in the Account Types List View.
    • Click Accounts > View Accounts from the Navigation menu, click the related account to view its Account Types sub-panel, and click Edit on the Actions dropdown.
  2. Update the Account Type information. See Field Descriptions for information about each field.
  3. Click Save to update the Account Type record.

Suppress one or more Account Type records
  1. In the Account Types List View, click the Selection box associated with the record(s) you want to suppress.
  2. Click Suppress.
    A confirmation box displays.
  3. Click OK.
    The Account Type records are suppressed.

You can also suppress a single Account Type record by clicking Suppress on the Account Types Detail View.

Manage records in a sub-panel

See Manage related information in sub-panels.

Update information related to multiple Account Types
  1. In the Account Types List View, click the Selection box associated with the records you want to update.
  2. Click Mass Update in the Actions dropdown.

For more information, see Editing and suppressing multiple records.

Duplicate Account Type information

Click Duplicate in the Account Types Detail View. Duplication is a convenient way of creating a similar Account Type. You can change and save the information in the duplicate record to create a new Account Type record.

Find and merge duplicate records

 Follow the instructions in Merging duplicate records.

Import Account Type records
  1. Click Import Account Types from the Navigation menu.
  2. Follow the instructions in Importing data.

Export records
  1. In the Account Types List View, click the Selection box associated with the records you want to export.
  2. Click Export in the dropdown and follow the instructions in Exporting data.

Field Descriptions

Fields that have an asterisk (*) are required; you must enter data in the field to save the Contact record.

Account Type*

Enter the Account Type of the Account Type. This is a mandatory field; you must enter an Account Type.

Account Category*

Click the Account Category to which this Account Type belongs from the Account Category Popup List View (displays upon clicking the Selection arrow) or the list of Account Categories matching the character(s) you enter. This is a mandatory field; you must click the Account Category.

Schedule

Click the section where this account type would appear in a financial plan.

Retirement Plan Type

Click the type of retirement plan this account type represents.

Beneficiary Required

Click the check box if this account type requires a beneficiary.

Report Sort

Enter an alphanumeric sort order for use in reports.

Description

Enter additional details about the account type.
