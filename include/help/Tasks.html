Overview

ProTracker enables you to create tasks and assign them to yourself or to other users. When you create a task, you can relate it to a specific record such as an account or an opportunity. You can also revise the task status and priority at any time to keep your records up-to-date.

Actions

Follow the instructions below to perform tasks such as creating, editing, and exporting Task records.

To…

Then…

Create a Task record
  1. Click Task on the Quick Create menu.
  2. Complete the Task Create Form. See Field Descriptions for information about each field. Fields that have an asterisk (*) are required; you must enter data in the field to save the record.
  3. Click Save to save the Task; click Cancel to return to the Task List View without saving your changes. To change the status of the task to Completed and create a new task, click Close and Create New. The status of the new task, by default, is set to Not Started.

Display a list of Tasks

Click NAMEs > View NAMEs.
The NAMEs List View displays.

Display the details for a NAME

Click the NAME name in the NAMEs List View.
The NAMEs Detail View displays.

Edit NAME information
  1. Access the NAMEs Edit View by doing one of the following:
    • Click Edit in the NAMEs Detail View.
    • Click the edit icon (pencil) associated with the NAME record in the NAMEs List View.
  2. Update the NAME information. See Field Descriptions for information about each field.
  3. Click Save to update the record.

Suppress one or more NAME records
  1. In the NAMEs List View, click the Selection box associated with the record(s) you want to suppress.
  2. Click Suppress.
    A confirmation box displays.
  3. Click OK.
    The records are suppressed.

You can also suppress a single NAME record by clicking Suppress on the NAMEs Detail View.

Manage records in a sub-panel

See Manage related information in sub-panels.

Update information related to multiple NAMEs
  1. In the NAMEs List View, click the Selection box associated with the records you want to update.
  2. Click Mass Update in the Actions dropdown.

For more information, see Editing and suppressing multiple records.

Duplicate NAME information

Click Duplicate in the NAMEs Detail View. Duplication is a convenient way of creating a similar NAME. You can change and save the information in the duplicate record to create a new record.

Find and merge duplicate records

 Follow the instructions in Merging duplicate records.

Import NAME records
  1. Click Import NAMEs from the Navigation menu.
  2. Follow the instructions in Importing data.

Export records
  1. In the NAMEs List View, click the Selection box associated with the records you want to export.
  2. Click Export in the dropdown and follow the instructions in Exporting data.

Field Descriptions

Fields that have an asterisk (*) are required; you must enter data in the field to save the record.

Related To
  1. Click the category from the dropdown.
  2. Click the record on the Popup List View (displays upon clicking the Selection arrow) or the list of records matching the character(s) you enter.

Assigned to

By default, the record is assigned to the creator of the Task. To change the assignment, click the user on the Users Popup List View (displays upon clicking the Selection arrow) or the list of users matching the character(s) you enter.

Subject*

Enter the subject of the Task. This is a mandatory field.

Activity Type

Click the category that best describes the Task.

Task Description

Enter a description of the task.

Project

Click the project on the Project Popup List View (displays upon clicking the Selection arrow) or the list of projects matching the character(s) you enter.

Status*

Click the current status of the task, such as Not Started, In Progress, and Completed from the dropdown. This is a mandatory field.

Attachment

Click Choose File to attach a document that is accessible to your computer, navigate to the location of the file, and select the file.

Priority*

Click a priority that reflects the importance of completing the task from the dropdown.

Start Date

On the Date Selector click the date the Task is to start and click the time in the adjoining dropdowns.

Enter the Start Date information if you want to add the Task on your calendar to receive a reminder when it is due.

Due Date

On the Date Selector click the date the Task is to be completed and click the time in the adjoining dropdowns.

Enter the Due Date information if you want to add the Task on your calendar to receive a reminder when it is due.

Client Task

Click the checkbox if this is a client task.

Assigned to Client

Click the checkbox if the Task has been assigned to a Client.

Results

Enter a brief description of the results of the task.

Teams

By default, your private team is assigned as the primary team. To change the primary team assignment, click the team on the Teams Popup List View (displays upon clicking the Selection arrow) or the list of teams matching the character(s) you enter.

To assign additional teams to the Task, click add (+) and click the team as described above. For more information on assigning teams, see Assigning Users and Teams to Records.

Workflow Related

Click a category (e.g., Client) and then click an item in the dropdown.

Private

Click the checkbox if this is a private task.

Date of Completion

On the Date Selector, click the date the task was completed.
