Overview

When a user creates a Contract record, the Types dropdown displays the contract types. Use the Contract Types module to define items in the Types dropdown of the Contracts module.

You can associate one or more documents with a Contract Type because a contract can reference multiple documents such as Statements of Work and Non-Disclosure Agreements. When users create a contract of a specific type, the associated documents display in the Documents sub-panel. Users can then edit and manage these documents as needed.

Availability

The Contact Types module is available to Administrators of the following editions:

  • Standard
  • Professional
  • Enterprise

Actions

Create a Contract Type

As an Administrator, use the following instructions to create a Contract Type.

  1. Click Contract Types on the Profile menu.
    The Contract Types List View displays.
  2. Click Create New.
    The Contract Types Create form displays.
  3. Complete the form by entering information for the fields described below. Fields that have an asterisk (*) are required; you must enter data in the field to save the Contract Type.
  4. Click Save.
    The Contract Type displays in the Contract Types List View.

Name*

Enter the name of the contract type. This is a mandatory field.

List Order*

Enter a number to specify the order in which the name is displayed in the Types dropdown in the Contracts module. This is a mandatory field.

Manage Contract Types

As an Administrator, use the following instructions to perform tasks such as exporting or suppressing Contact Type records.

To…

Then…

Sort the Contract Types records in the Contract Types List View

Click on any column title; to reverse the sort order, click the column title again.

Display the details of a Contract Type record

Click the name of the contract type in the Contract Types List View.

Edit the Contract Type record
  1. Click the edit (pencil) icon associated with the Contract Type you want to update in the Contract Type List View.
    The Contract Types Edit View displays.
  2. Update the information using the field descriptions in Create a Contract Type.
  3. Click Save.
    The Contract Type record is updated.

Suppress multiple Contract Type records
  1. In the Contract Types List View, click the selection box associated with the records you want to suppress.
  2. Click Suppress.
    A confirmation box displays.
  3. Click OK.
    The records are suppressed.

For more information, see  Editing and suppressing multiple records.

Alternatively, to suppress a single record, you can click Suppress on the Contract Types Detail View.

Export records
  1. In the Contract Types List View, click the selection box associated with the records you want to export.
  2. Click Export in the dropdown and follow the instructions in Exporting data.

Associate Documents with a Contract Type

As an Administrator, use the following instructions to associate documents with a Contract Type record.

  1. Click Contract Types on the Profile menu.
    The Contract Types List View displays.
  2. Click the name of the Contract Type with which you want to associate a document.
    The Contract Type Detail View displays.
  3. Click Select in the Documents sub-panel.
    The Documents Popup List View displays.
  4. Click the Document you want to associate to the Contract Type.
    The Contract Types Detail View displays with the document listed in the Documents sub-panel.

To remove the Document record, click remove on the Actions dropdown associated with the Contract Types record.
