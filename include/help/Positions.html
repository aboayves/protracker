Overview

Use the Positions module to record the specific portfolio holdings of an account or client and review holdings across all clients. Due to the large volume of data involved, positions are generally imported, not manually entered.

Actions

Follow the instructions below to perform tasks such as creating, editing, and exporting Position records.

To…

Then…

Create a Position record
  1. Access the Position Create form using one of the following:
    • Click Accounts > Positions > Create Positions from the Navigation menu.
    • Click Create New on the Positions List View.
  2. Complete the Position Create Form. See Field Descriptions for information about each field. Fields that have an asterisk (*) are required; you must enter data in the field to save the record.
  3. Click Save to save the record and view its detail page, or click Cancel .

Display a list of Positions

Click Accounts > Positions > View Positions.
The Positions List View displays.

Display the details for a position

Click the position name in the Positions List View.
The Position Detail View displays.

Edit position information
  1. Access the Position Edit View by doing one of the following:
    • Click Edit in the Position Detail View.
    • Click the edit icon (pencil) associated with the Position record in the Positions List View.
  2. Update the position information. See Field Descriptions for information about each field.
  3. Click Save to update the record.

Suppress one or more Position records
  1. In the Positions List View, click the Selection box associated with the record(s) you want to suppress.
  2. Click Suppress.
    A confirmation box displays.
  3. Click OK.
    The records are suppressed.

You can also suppress a single NAME record by clicking Suppress on the Position Detail View.

Update information related to multiple positions
  1. In the Positions List View, click the Selection box associated with the records you want to update.
  2. Click Mass Update in the Actions dropdown.

For more information, see Editing and suppressing multiple records.

Duplicate position information

Click Duplicate in the Position Detail View. Duplication is a convenient way of creating a similar position. You can change and save the information in the duplicate record to create a new record.

Find and merge duplicate records

 Follow the instructions in Merging duplicate records.

Import Position records
  1. Click Accounts > Positions > Import Positions from the Navigation menu.
  2. Follow the instructions in Importing data.

Export records
  1. In the Positions List View, click the Selection box associated with the records you want to export.
  2. Click Export in the dropdown and follow the instructions in Exporting data.

Field Descriptions

Fields that have an asterisk (*) are required; you must enter data in the field to save the record.

Account

Click the Account related to the Position in the Accounts Popup List View (displays upon clicking the Selection arrow) or the list of Accounts matching the character(s) you enter.

Source

If applicable, enter the name of the portfolio management system or website that provided the data.

Account Number

Security Type

Click the type of security this holding represents; e.g., cash, equities, or mutual funds.

Position

Enter the name of the holding.

Value

Enter the total dollar value of this holding. (Quantity * Price)

Symbol

Enter the ticker symbol used in trading. Clicking on the Symbol in a list view or sub-panel will open a Google Finance page with more information about this holding.

Quantity

Enter the total number of shares. For cash positions, this is the same as the total dollar amount held in cash.

Price

Enter the price of a single share of this holding. For cash positions, the price is 1.00.

Price Date

Click on the Date Selector the date on which this position was valid.

Position Date

Click on the Date Selector the date ??.

Description

Enter additional details about this Position record.
