Overview

Use the Positions module to record the specific portfolio holdings of an account or client and review holdings across all clients. Due to the large volume of data involved, positions are generally imported, not manually entered.

Availability

The Positions module is available to users of the following editions:

  • Standard
  • Professional
  • Enterprise

Actions

Create a Position record

To create a Position record follow these instructions.

  1. Access the Positions Create form by doing one of the following:
    • Click Accounts > Positions > Create Positions on the Navigation menu.
    • Click Accounts > View Accounts from the Navigation menu, click the related Account to view its Positions sub-panel, and then click Create on the Positions sub-panel Actions dropdown.
  2. Enter information for the fields described below.
  3. Click Save to create the record; click Cancel to return to the List View without saving the record.

Account

Click the Account related to the Position in the Accounts Popup List View (displays upon clicking the Selection arrow) or the list of Accounts matching the character(s) you enter.

Source

If applicable, enter the name of the portfolio management system or website that provided the data.

Account Number

Security Type

Click the type of security this holding represents; e.g., cash, equities, or mutual funds.

Position

Enter the name of the holding.

Value

Enter the total dollar value of this holding. (Quantity * Price)

Symbol

Enter the ticker symbol used in trading. Clicking on the Symbol in a list view or sub-panel will open a Google Finance page with more information about this holding.

Quantity

Enter the total number of shares. For cash positions, this is the same as the total dollar amount held in cash.

Price

Enter the price of a single share of this holding. For cash positions, the price is 1.00.

Price Date

Click on the Date Selector the date on which this position was valid.

Position Date

Click on the Date Selector the date ??.

Description

Enter additional details about this Position record.

Edit a Position record

Use the following instructions to update a Position record.

  1. Access the Position Edit View by doing one of the following:
    • In the Navigation menu, click Accounts > Positions > View Positions, and click the edit (pencil) icon associated with the record you want to change.
      The Positions Edit View displays.
    • In the Navigation menu, click Accounts > Accounts > View Accounts , click the Account, and then click Edit on the Position sub-panel Action dropdown.
  2. Update information for the fields described in Create a Position record.
  3. Click Save to update the record; click Cancel to return to the Positions List View without saving the record.

Suppress a Positions record

Suppress Position record(s) using the Positions List View. For more information, see Editing and suppressing multiple records.

  1. In the Navigation menu, click Accounts > Positions > View Positions.
    The Positions List View displays.
  2. In the Positions List View, click the selection box associated with the record(s) you want to suppress.
  3. Click Suppress.
    A confirmation box displays.
  4. Click OK.
    The records are suppressed.

Alternatively, to suppress a single record, you can click Suppress on the Positions Detail View.
