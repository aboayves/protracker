Overview

Use the Trust Amendments module to record changes to a trust.

Actions

Follow the instructions below to perform tasks such as creating and editing Trust Amendment records.

To…

Then…

Create a Trust Amendment record
  1. Click Estate Planning > Trusts > View Trusts on the Navigation menu.
    The Trusts List View displays.
  2. Click the Trust record.
    The Trust Detail View displays.
  3. Click Create on the Trust Amendment sub-panel Actions dropdown.
    The Trust Amendment Create Form displays.
  4. Complete the Trust Amendment Create Form. See Field Descriptions for information about each field. Fields that have an asterisk (*) are required; you must enter data in the field to save the record.
  5. Click Save to save the record and view its detail page, or click Cancel .

Edit Trust Amendment information
  1. Click Estate Planning > Trusts > View Trusts on the Navigation menu.
    The Trusts List View displays.
  2. Click the Trust record.
    The Trust Detail View displays.
  3. Click Edit on the Trust Amendments sub-panel Actions dropdown.
    The Trust Amendment Edit Form displays.
  4. Update the information. See Field Descriptions for information about each field.
  5. Click Save to update the record.

Suppress a Trust Amendment record

Click Suppress on the Trust Amendment sub-panel Actions dropdown.

Field Descriptions

Fields that have an asterisk (*) are required; you must enter data in the field to save the record.

Name

Describe the trust amendment.

Assigned to

By default, the record is assigned to the creator of the Trust record. To change the assignment, click the user on the Users Popup List View (displays upon clicking the Selection arrow) or the list of users matching the character(s) you enter.

File*

Click Choose File and navigate to and select your trust file.
The file name displays in the Select File box.

Teams *

By default, your private team is assigned as the primary team. To change the primary team assignment, click the team on the Teams Popup List View (displays upon clicking the Selection arrow) or the list of teams matching the character(s) you enter.

To assign additional teams to the Trust, click add (+) and click the team as described above. For more information on assigning teams, see Assigning Users and Teams to Records.

Publish Date

On the Date Selector click the date the trust was published.

Category

Click a category.

Sub Category

Click a sub category.

Description

Enter additional details about the trust amendment.
