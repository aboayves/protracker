Keyboard Shortcuts

Overview

Overview

This article lists keyboard shortcuts you can use with ProTracker.

Global Shortcuts

  • Alt + 0: Puts the cursor in the first global search input
  • Alt + H: Opens the help file

From within a record's Detail View

  • Alt + I: Edit
  • Alt + U: Duplicate
  • Alt + D: Delete

From within a record's Edit View

  • Alt + S: Save
  • Alt + L: Cancel
  • Alt + 1: Puts the cursor in the first input on the Edit form

From within a module's List View

  • Alt + 1: Puts teh cursor int eh first search form input
  • Alt + A: Advanced Search Link

To open a module or screen in a new window or tab (Firefox, IE, and Chrome): Press CTRL, then click the link for the item you want to open in a new window or tab
