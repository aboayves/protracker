Overview

Use the Prices module to record the prices of securities to calculate a client's current positions. Prices are generally imported using price files, not manually entered. If you import positions and that data includes price information, importing price data separately is not necessary. Only users with database administration privileges can view and edit Prices.

Availability

The Prices module is available to users of the following editions:

  • Standard
  • Professional
  • Enterprise

Actions

Create a Price record

To create a Price record follow these instructions.

  1. In the Profile menu click Admin > Prices.
    The Prices List View displays.
  2. Click Create New.
    The Prices Create form displays.
  3. Enter information for the fields described below.
  4. Click Save to create the record; click Cancel to return to the Prices List View without saving the record.

Symbol

Enter the ticker symbol used in trading. Clicking on the Symbol in a list view or sub-panel will open a Google Finance page with more information about this holding.

Price Date

Click on the Date Selector the date on which this price was valid.

Price

Enter the price of a single share of this holding.

Edit a Price record

Use the following instructions to update a Price record.

  • In the Profile menu click Admin > Prices.
    The Prices List View displays.
  • Click the edit (pencil) icon associated with the record you want to change.
    The Prices Edit View displays.
  • Update information for the fields described in Create a Price record.
  • Click Save to update the record; click Cancel to return to the Price List View without saving the record.

    • Suppress a Price record

    Suppress Price record(s) using the Prices List View. For more information, see Editing and suppressing multiple records.

    1. In the Profile menu click Admin > Prices.
      The Prices List View displays.
    2. In the Prices List View, click the selection box associated with the record(s) you want to suppress.
    3. Click Suppress.
      A confirmation box displays.
    4. Click OK.
      The records are suppressed.

    Alternatively, to suppress a single record, you can click Suppress on the Prices Detail View.
