Overview

Use the Activity Types module to categorize calls, meeting, notes, and tasks. All users can view activity types. Only users with database administration privileges can edit activity types.

Availability

The Activity Types module is available to users of the following editions:

  • Standard
  • Professional
  • Enterprise

Actions

Create an Activity Type

  1. Click Calls > View Activity Types on the Navigation menu.
    The Activity Types List View displays.
  2. Click Create New.
    The Activity Types Create form displays.
  3. Complete the form by entering information for the fields described below. Fields that have an asterisk (*) are required; you must enter data in the field to save the Activity Type.
  4. Click Save.
    The Activity Type record is added.

Activity Type*

Enter the name of the Activity Type. This is a mandatory field; you must enter a name.

Report Sort

Enter an alphanumeric sort order for use in reports.

Description

Enter additional details about the activity type.

Assigned to

By default, the record is assigned to the creator of the Activity Type record. To change the assignment, click the user on the Users Popup List View (displays upon clicking the Selection arrow) or the list of users matching the character(s) you enter.

Teams*

By default, your private team is assigned as the primary team. To change the primary team assignment, click the team on the Teams Popup List View (displays upon clicking the Selection arrow) or the list of teams matching the character(s) you enter.

To assign additional teams to the Activity Type, click add (+) and click the team as described above. For more information on assigning teams, see Assigning Users and Teams to Records.

Edit an Activity Type

Use the following instructions to edit an Activity Type.
  1. Click Calls > View Activity Types from the Navigation menu.
    The Activity Types List View displays.
  2. Click the edit (pencil) icon on the record you want to edit.
    The Activity Types Edit View displays.
  3. Complete the form by entering information for the form fields as described in Create an Activity Type.
  4. Click Save.
    The updates are saved to the Activity Types record.

Suppress an Activity Type

Use the following instructions to suppress an Activity Type.
  1. Click Calls > View Activity Types from the Navigation menu.
    The Activity Types List View displays.
  2. Click the selection box associated with the record(s) you want to suppress.
  3. Click Suppress in the Actions dropdown.
    The confirmation box displays.
  4. Click OK.
    The record(s) are suppressed.
