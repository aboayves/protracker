Overview

Use the Account Categories module to define broad categories of accounts for report grouping and analysis, such as “Investment Assets” or “Liabilities”. All users can view account categories. Only users with database administration privileges can edit account categories.

Availability

The Account Categories module is available to users of the following editions:

  • Standard
  • Professional
  • Enterprise

Actions

Create Account Categories

Use these instructions to access the Account Categories Create form to create Account Categories.

  1. Click Accounts > Account Categories > Create Account Categories from the Navigation menu.
    The Account Categories Create form displays.
  2. Enter the following information in the form. Fields that have an asterisk (*) are required; you must enter data in the field to save the Account Categories.

Account Category*

Enter the name of the Account Category. This is a mandatory field.

Description

Enter a brief description of the Account Category.

Liability

Click this checkbox if this Account Category type is a liability.

Report Sort

Enter the alphanumeric sort order for this category. This is used for reporting.

Assigned to

By default, the record is assigned to the creator of the Account Categories record. To change the assignment, click the user on the Users Popup List View (displays upon clicking the Selection arrow) or the list of users matching the character(s) you enter.

Click Save to save your changes; click Cancel to go to the Account Categories List View without saving your changes.

Manage Account Categories

Use the following instructions to perform tasks such as editing and suppressing Account Categories.

To…

Then…

Edit the Account Category
  1. Click Accounts > Account Categories from the Navigation menu.
    The Account Categories List View displays.
  2. Click the edit icon (pencil) next to the Account Category you want to edit.
    The Account Categories Quick form displays.
  3. Make the changes to the information in the form.
  4. Click Save.

Suppress
  1. In the Account Categories List View, click the selection box associated with the records you want to suppress.
  2. Click Suppress.

A confirmation box displays.

  1. Click OK.

The Account Categories are suppressed.

Alternatively, you can click Suppress on the Account Categories Details View.

Note: Before you suppress Account Categories, reassign any Account Types that belong to the suppressed Account Category to a different Account Category.

Update information related to multiple Account Categories
  1. In the Account Categories List View, click the selection box associated with the records you want to update.
  2. Click Mass Update in the dropdown.

For more information, see  Editing and suppressing multiple records.

Export records
  1. In the Account Categories List View, click the selection box associated with the records you want to export.
  2. Click Export in the dropdown and follow the instructions in Exporting data.

Merge duplicate records
  1. In the Account Categories List View, click the selection box associated with the records you want to merge.
  2. Click Merge in the dropdown and follow the instructions in Merging duplicate records.
