Overview

Use the Contact Types module to categorize contact records. All users can view Contact types.

Only users with database administration privileges can edit Contact types.

Availability

The Contact Types module is available to users of the following editions:

  • Standard
  • Professional
  • Enterprise

Actions

Create a Contact Type

As an Administrator, use the following instructions to create a Contact Type.

  1. Click Contacts > View Contact Types on the Navigation menu.
    The Contact Types List View displays.
  2. Click Create New.
    The Contact Types Create form displays.
  3. Complete the form by entering information for the fields described below. Fields that have an asterisk (*) are required; you must enter data in the field to save the Contact Type.
  4. Click Save.
    The Contact Type displays in the Contact Types List View.

Contact Type*

Enter the name of the contact type. This is a mandatory field.

Report Sort

Enter an alphanumeric code for use in sorting records for reports.

Description

Enter additional details about the contact type.

Assigned to

By default, the record is assigned to the creator of the Contact Type record. To change the assignment, click the user on the Users Popup List View (displays upon clicking the Selection arrow) or the list of users matching the character(s) you enter.

Edit a Contact Type

As an Administrator, use the following instructions to edit a Contact Type.

  1. Click Contacts > View Contact Types on the Navigation menu.
    The Contact Types List View displays.
  2. Click the edit (pencil) icon associated with the Contact Type you want to update.
    The Contact Types Edit View displays.
  3. Update the information using the field descriptions in Create a Contact Type.
  4. Click Save.
    The Contact Type record is updated.

Suppress a Contact Type

As an Administrator, use the following instructions to suppress a Contact Type.

  1. Click Contacts > View Contact Types on the Navigation menu.
    The Contact Types List View displays.
    1. Click the selection box associated with the record(s) you want to suppress.
    2. Click Suppress in the Actions dropdown.
      The confirmation box displays.
    3. Click OK.
      The record(s) are suppressed.

    2. Sub-panels/Related Information

    None.
