Overview

The Contracts module lists all existing contracts and associated information such as the account name, contract status, and the primary team assigned to the contract. When you create a contract, it is automatically listed under the related account. It is also listed under the related opportunity and quote, if you specified one.

A contract can reference multiple documents such as a Master Service Agreement (MSA) and one or more Statements of Work (SOW). The details of any document that is referenced in a contract displays in the Documents sub-panel. Other information such as notes, contact information for individuals involved in managing the contract, product information, and quotes are also displayed in sub-panels on the contract's detail page.

Notes: By default, this module is hidden. The administrator must display it in order to enable you to use it.

Availability

The Documents module is available to users of the following editions:

  • Standard
  • Professional
  • Enterprise

Actions

Create a Contract

To create a Contract follow these instructions.

  1. In the Navigation menu, click Billing > Contracts > Create Contract.
    The Contracts Create form displays.
  2. Enter information for the fields described below. Fields that have an asterisk (*) are required; you must enter data in the field to save the document.
  3. Click Save to create the record; click Cancel to return to the List View without saving the record.

Contract Name*

Enter a name for the Contract. This is a mandatory field.

Status*

Click the status of the Contract; e.g., Not Started, In Progress, or Signed. This is a mandatory field.

Reference Code

Enter the internal reference number for the Contract.

Start Date

Click on the Date Selector the date on which the Contract takes effect.

Client name*

Click the Client that this Contract is related to on the Clients Popup List View (displays upon clicking the Selection arrow) or the list of Contracts matching the character(s) you enter. This is a mandatory field.

End Date

Click on the Date Selector the date on which the Contract ends.

Opportunity Name

Enter the opportunity to which this Contract is related.

Type

Click the type of Contract; e.g., Retainer.

Client Signed Date

Click on the Date Selector the date on which the client signed the contract.

Currency

Click the currency from the dropdown.

Company Signed Date

Click on the Date Selector the date on which your organization signed the contract.

Contract Value

Enter the amount specified in the contract.

Expiration Notice

Click on the Date Selector the date and click the time to send a reminder to the Assigned User that the contract will expire shortly.

Assigned to

By default, the record is assigned to the creator of the Contract record. To change the assignment, click the user on the Users Popup List View (displays upon clicking the Selection arrow) or the list of users matching the character(s) you enter.

Contract Value (USD)

Billing Type

Click the type of billing; e.g., retainer or fixed fee.

Attach Document

Click Choose File and navigate to and select your Contracts file.
The file name displays in the Select File box.

Description

Enter additional information about the Contract.

Teams*

By default, your private team is assigned as the primary team. To change the primary team assignment, click the team on the Teams Popup List View (displays upon clicking the Selection arrow) or the list of teams matching the character(s) you enter. This is a mandatory field.

To assign additional teams to the Contract, click add (+) and click the team as described above. For more information on assigning teams, see Assigning Users and Teams to Records.

Manage Contracts

Use the following instructions perform tasks such as viewing, editing, or suppressing Contract records.

To…

Then…

View the list of Contracts

Click Billing > Contracts > View Contracts on the Navigation menu.
The Contracts List View displays.

Sort the Contracts records

Click on any column title; to reverse the sort order, click the column title again.

Display the details of a Contract record

Click the name of the contract in the Contracts List View.

Edit a Contract record

  1. In the Contracts List View, click the edit (pencil) icon associated with the record you want to change.
    The Contracts Edit View displays.
  2. Update the information using the field descriptions in Create a Contract.
  3. Click Save.
    The Contract Type record is updated.

Suppress multiple Contract records

  1. In the Contracts List View, click the selection box associated with the records you want to suppress.
  2. Click Suppress.
    A confirmation box displays.
  3. Click OK.
    The records are suppressed.

For more information, see  Editing and suppressing multiple records.

Alternatively, to suppress a single document, you can click Suppress on the Contracts Detail View.

Update information related to multiple

  1. In the Contracts List View, click the selection box associated with the records you want to update.
  2. Click Mass Update in the dropdown.

For more information, see  Editing and suppressing multiple records.

Duplicate a Contract record

Click Duplicate on the Contracts Detail View Actions dropdown. Duplication is a convenient way of creating a similar Contract record. You can change the information in the duplicate record to create a new Contract.

Create or manage related records in sub-panels

See Managing related information in sub-panels.

Import records

  1. Click Billings > Contracts > Import Contracts
    Step 1: Upload Import File form displays.
  2. Follow the instructions in Importing data.

Export records
  1. In the Contracts List View, click the selection box associated with the records you want to export.
  2. Click Export in the dropdown and follow the instructions in Exporting data.
